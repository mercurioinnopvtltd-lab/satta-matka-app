# Satta Matka Application

**Complete Full-Stack Satta Matka Gaming Platform**

## ⚠️ LEGAL DISCLAIMER
This application is for educational purposes only. Satta Matka is illegal in most jurisdictions. Users assume full legal responsibility.

## Features

### Admin Panel
- User Management (Admin, Agent, Player)
- Game Management & Results
- Payment Verification
- Commission Management
- Reports & Analytics
- Audit Logs

### Agent Dashboard
- Player Management
- Commission Tracking
- Withdrawal Requests
- Performance Reports
- Bets Management

### Player App
- User Registration & Login
- Wallet Management
- Place Bets
- View Results
- Payment Methods
- Betting History
- Winnings Tracker

## Tech Stack

- **Backend**: Node.js, Express.js
- **Frontend**: React.js
- **Database**: MongoDB
- **Authentication**: JWT
- **Payments**: Stripe/Razorpay
- **Hosting**: Docker, AWS/Heroku

## Quick Start

```bash
# Clone repository
git clone https://github.com/mercurioinnopvtltd-lab/satta-matka-app.git

# Start with Docker
docker-compose up

# Visit http://localhost:3000
```

## Installation

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```
